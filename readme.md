## **Future Supported Instructions**

| Type | Instruction | Description |
| :--- | :--- | :--- |
| **Arithmetic** | `ADD`, `SUB` | Basic integer addition and subtraction. |
| | `ADDI` | Add immediate (e.g., incrementing counters). |
| | `SLT` | Set Less Than (Signed comparison). |
| **Logic** | `AND`, `OR`, `XOR` | Bitwise operations for masking and flags. |
| | `ANDI`, `ORI`, `XORI` | Immediate bitwise operations. |
| **Shift** | `SLL`, `SRL` | Logical shifts (efficient multiplication/division). |
| | `SRA` | Arithmetic Right Shift (preserves sign bit). |
| | `SLLI`, `SRLI`, `SRAI` | Immediate shifts for fixed operations. |
| **Memory** | `LW` | Load Word (32-bit) from RAM. |
| | `SW` | Store Word (32-bit) to RAM. |
| **Branch** | `BEQ`, `BNE` | Branch if Equal / Not Equal. |
| | `BLT`, `BGE` | Branch if Less Than / Greater or Equal (Signed). |
| **Control** | `JAL` | Jump And Link (Function Call). |
| | `JALR` | Jump And Link Register (Return/Computed Jump). |
